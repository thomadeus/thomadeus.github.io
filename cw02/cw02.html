Exercise 6

HTML will provide the basic structure of the site/page, which can be further enhanced visually and aurally with CSS. I believe their should be an underlying structure to the site/pages but am also a fan of allowing departments, teachers, instructors and so on to have some ability to customize their respective pages aesthetics i.e heavy use of external style sheets

Tags To Use

  1. <div>
    Helps separate the different parts and components of a web page. Useful for grouping gvarious HTML tags so that the relevant setions can be created and style can be applied to them
  2. <h1, h2, h3, ... h6>
    Headings highlight the important topic and structure of the page. Used to create headings in descending order of importantance where h6 is least important. So teachers could place a h1 heading for assignment due dates at the top of the page, indiciating that information is most important.
  3. <p>
    The paragraph tags is used to define paragraph content in the html document. Internet browsers automatically add white space before and after blocks of text contained in the pargraph tag
  4. <ul> & <ol>
    List tags are a must for anyone wanting to display information as a list. Unordered list items can be displayed in bullet points using the <ul >tag. Ordered items, where the content of the data has some inherent order, can be displayed in ascending or descending order using the <ul > tag
  5. <a>
    Employees will definitely want to prove links on the website. The anchor tag is used to define the beginning and end of a hyperlink text. Useful for linking from one page to another.
  6. <img>
    Used to embed an image to an HTML page. Useful for displaying pictures of teachers, students, maps, school events, and so on.
  7. <table>
    A block element that is used to create a table. So if teachers want to represent some data in a tabular layout, this is necessary. For example, a calendar with columns containing a day of the week, and and the cells in rows can contain information about upcoming exams or assignments.
  8. <hr>
    The horizontal rule tag inserts a thematic break in an html page. Useful for visually dividing/separating sections of a document.
  9. <b>
    The bold tag can be used to further emphasize important textual information.
  10. <br>
    This tag will show a blank line. Good for breaking up sections in pages that arent paragraphs.
